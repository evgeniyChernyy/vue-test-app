## LOCAL DEVELOPMENT

## Setup project
```
npm install
```

### Run server
```
node server.js
```

### Compile for development
```
npm run serve
```
The application will be available at the URL specified in the message


## DEPLOY ON REAL SERVER

### Setup project
```
npm install
```

### Build frontend
```
npm run build
```

### Run server
```
node server.js
```
